# pico8-miyoo-flip
Pico8 standalone splore for Miyoo Flip console

https://www.lexaloffle.com/pico-8.php

## Installation:

Clone or donwload de repository and put the "App" and "Emu" folder in the root of SDCARD.

Download the pico8 binary from lexaloffle web site and get the raspberry pi version, them put pico8_64 and pico8.dat in:

App/Pico8/bin/

enjoy!!

## controls:

Menu = escape.
L1 = pagup.
R1 = pagdown.
select = switcher mouse mode.

in mouse mode:

L1 = mouse button left.
R1 = mouse button right.
dpad = mouse cursor.
